В этом проекте реализованы два задания на языке Python, выполненные в среде Jupyter Notebook.

1. **Задание 1**  
   Решение задачи, где дана строка из латинских букв. Программа выводит:  
   - среднюю букву, если длина строки нечётная;  
   - две средние буквы, если длина строки чётная.

2. **Задание 2**  
   MVP dating-сервис, в котором даны два списка: мальчики и девочки.  
   Программа сортирует имена по алфавиту и формирует идеальные пары по индексам.  
   Если списки разной длины, выводится предупреждение, что кто-то может остаться без пары.

## Технологии и окружение

- Язык программирования: Python 3
- Среда разработки: Jupyter Notebook

## Как использовать

1. Откройте соответствующий notebook в Jupyter Notebook.  
2. Запустите ячейки с кодом для выполнения каждой задачи.  
3. При необходимости измените входные данные (слова или списки имен) под свои варианты.  
4. Программа выведет результат в консоли.
