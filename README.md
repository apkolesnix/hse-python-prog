Описание задания
Цель задания — научиться создавать и использовать функции в Python. Реализованы две функции согласно требованиям:

sum_distance(from_, to_)
Функция принимает два целых числа — границы диапазона. Если границы заданы в обратном порядке, они меняются местами. Возвращает сумму всех чисел в этом диапазоне включительно.

trim_and_repeat(string, offset=0, repetitions=1)
Функция принимает строку и параметры offset и repetitions. Обрезает строку слева на offset символов, затем повторяет полученную строку repetitions раз. Возвращает результирующую строку.

Как проверить
В ноутбуке с кодом есть примеры вызова функций для проверки правильности работы.

Примеры демонстрируют разные варианты параметров.

Функции работают корректно и не вызывают ошибок.
