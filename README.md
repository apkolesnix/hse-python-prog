## Тема: Функции и области видимости

---

### Описание задания  
Выполнить практическое задание на написание функций в Python.  
Цель — научиться создавать функции с параметрами и возвращать результаты.

---

### Реализованные функции

1. **sum_distance(from_, to_)**  
- Принимает два целых числа: `from_` и `to_`.  
- Если `from_` больше `to_`, значения меняются местами.  
- Возвращает сумму всех целых чисел в диапазоне от `from_` до `to_` включительно.

2. **trim_and_repeat(string, offset=0, repetitions=1)**  
- Принимает строку `string`.  
- Параметр `offset` определяет, сколько символов обрезать слева (по умолчанию 0).  
- Параметр `repetitions` задаёт количество повторений результирующей строки (по умолчанию 1).  
- Возвращает новую строку, обрезанную и повторённую согласно параметрам.
